## 动态规划

### 值域 dp

- [2026牛客寒假算法基础集训营6 H](https://ac.nowcoder.com/acm/contest/120566/H) （该比赛需付费参加）

模型简化：给定长度为 $n$ $(1\le n\le 2048)$ 的数组 $a$ 和 $b$ $(0\le a_i, b_i< 2048)$，给定一个初值为 $0$ 的变量 $x$，你需要执行恰好 $n$ 次操作，第 $i$ $(1\le i\le n)$ 次操作必须从下面的两个操作中选一个执行：

- $x := \max(0, x - a_i)$
- $x := x\oplus b_i$

其中 $\mathtt{:=}$ 表示赋值符号。求执行完恰好 $n$ 次操作后 $x$ 的最大值。

### 数据结构 dp

- [Codeforces Round 1082 (Div. 2) C2](https://codeforces.com/contest/2202/problem/C2)

模型简化：设 $x$ 是一个长度为 $n$ 的序列，给定一个整数 $k$ $(k\ge 0)$ 你需要对 $x$ 执行如下的操作，使序列 $x$ 的长度恰好为 $n + k$ : 

- 如果 $k = 0$，那么不需要进行任何操作。
- 否则，选择任意一个下标 $1\le i\le |x|$，将 $(x_i + 1)$ 插入至 $x_i$ 右侧的相邻位置。
- 重复操作 $2$ 直到序列 $x$ 的长度恰好为 $n + k$ 。

定义函数 $f(b)$ 表示能通过上述操作产生序列 $b$ 的最短序列的长度，给定一个长度为 $n$ $(1\le n\le 3\times10^5, 1\le a_i\le 10^9)$ 的数组 $a$，求 $\sum_{i = 1}^{n}\sum_{j = i}^{n} f(a[i, j])$，其中 $a[i, j]$ 表示数组 $[a_i, a_{i + 1}, \cdots, a_{j - 1}, a_j]$ 。

评价：很不错的 dp 好题，中间用到了线段树进行区间赋值与维护区间和，具备一定的思维量。

- [Codeforces Round 426 (Div. 1) B](https://codeforces.com/contest/833/problem/B)

模型简化：给定一个长度为 $n$ 的数组 $a$ $(1\le n\le 3.5\times 10^4, 1\le a_i\le n)$ 和一个整数 $k$ $(1\le k\le \min(50, n))$，定义一个子段的权值为子段内元素的种类数，求把 $a$ 划分为恰好 $k$ 段时 $k$ 个子段的权值和的最大值。

评价：很好的数据结构维护 dp 题，使用线段树维护 dp 数组并进行滚动更新。
